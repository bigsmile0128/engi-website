# Engi Website

The Engi website built by the Engi network

## Development

```
npm install
npm start
```

## Production Preview

Create the production build.

```
npm run build
```

Install `serve` for serving static files.

```
npm i -g serve
```

Serve the locally built files.

```
serve -s build
```

## Mock server

```
npm run server
```

## Storybook

```
npm run storybook
```
